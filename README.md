Welcome!
Welcome to the official web project for Bent Cypress Homestead. The name comes from the idea that in nature, and in life, the things that bend are the ones that don't break. Our homestead is a constant adventure in learning to be resilient, resourceful, and work with the beautiful chaos of nature.

This website is a living document of that journey. It's a place to share our stories, photos, and the lessons we learn along the way. It's also a personal project for me to practice and sharpen my skills in web development.

What You'll Find Inside
🏡 Homepage: A landing page to welcome visitors and showcase the latest happenings.

📝 Blog: Stories and updates from the homestead, like our adventures in managing Springtime Chaos and Chicks.

🖼️ Photo Gallery: A visual journey through the seasons, the animals, and the landscape of our home.

🛠️ Tech Stack & Learnings
This site is built with fundamental web technologies, with a focus on clean, modern practices. This project is a key part of my goal to "Pound the Basics" of IT and development.

HTML5: For the structure and content.

Tailwind CSS: For styling the site with a utility-first approach. The site currently uses the Tailwind CDN for easy setup.

JavaScript: For interactive elements, like the subtle fade-in effects you see as you scroll.

🚀 Running This Project Locally
Since this is a static website and uses the Tailwind CSS link directly, there's no complex build process required.

Clone the repository:

Bash

git clone <your-repository-url>
Navigate to the folder:

Bash

cd <repository-name>
Open index.html in your browser:
You can simply double-click the index.html file to open it.

Pro Tip: For the best development experience, I recommend using the Live Server extension in a code editor like VS Code. It will automatically reload the page in your browser every time you save a file, which saves a ton of time.
